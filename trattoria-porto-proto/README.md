===

## Setup

```bash
npm i

# ./.nuxt/tsconfig.json を生成 -> ./tsconfig.json に読み込みさせるため
npm run build
```

## Development

```bash
npm run dev
```

## Production

Build the application for production:

```bash
npm run build
```

Checkout the [deployment documentation](https://v3.nuxtjs.org/docs/deployment).
